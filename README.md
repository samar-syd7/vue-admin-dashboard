# Vue Admin Dashboard

Live Demo: https://69618d7a19401d4c6d68cbfa--heartfelt-valkyrie-2e7639.netlify.app

A simple Vue 3 admin dashboard demonstrating real-world features:
- Login & route protection
- Dashboard layout
- API data table
- CRUD operations using dialogs
- Loading and error handling

Tech Stack:
- Vue 3
- Vue Router
- REST API (Axios)
- Vuetify / Tailwind
